# Predicting Startup Outcomes



# Project Overview

Startups operate in environments with **high uncertainty and risk**. Many fail within their early years despite receiving significant funding.

This project analyzes **startup data to identify patterns that influence success, failure, and investment outcomes.**

Using data visualization and exploratory analysis, the project focuses on:

* 📉 Startup failure patterns
* 📈 Funding vs survival relationships
* 🌍 Regional startup ecosystem performance
* 💰 Return on Investment (ROI) categories

The goal is to provide **insights for investors, analysts, and entrepreneurs** to better understand startup dynamics.

---

# Project Objectives

✔ Analyze **startup survival rates**
✔ Identify **factors influencing startup failure**
✔ Visualize **funding impact on outcomes**
✔ Compare **regional startup performance**
✔ Explore **ROI categories across startups**

# Key Metrics Analyzed

The dashboard evaluates several critical startup indicators:

* **Startup Survival Rate**
* **Startup Failure Rate**
* **Average Operating Startup Age**
* **Funding vs Closure Ratio**
* **Total Funding Distribution**
* **Regional Startup Performance**
* **ROI Outcome Categories**

These metrics help uncover **hidden patterns in startup success and failure.**

---

# 📊 Dashboard Visualizations

## Startup Survival Overview

## Funding vs Closure Trends

---

## Funding Impact on ROI

Explores how **funding levels influence ROI categories and startup longevity.**

# 🔍 Key Insights

 Many startups fail within the **first few operational years**.

 Higher funding does **not always guarantee startup success**.

 Some regions demonstrate **stronger startup ecosystems and survival rates**.

 ROI outcomes vary significantly depending on **funding size and operational lifespan**.

# Project Summary

This project provides a data-driven analysis of startup ecosystems by examining funding patterns, survival rates, and investment outcomes.

Through interactive dashboards and exploratory analysis, the project highlights how factors such as **funding levels, operational lifespan, and geographic location** influence startup success or failure.

The insights from this analysis can help:

• Investors evaluate potential startup risks  
• Entrepreneurs understand market dynamics  
• Analysts identify patterns in startup growth and failure  

By transforming raw startup data into **clear visual insights**, this project demonstrates the value of **data analytics and business intelligence in decision-making**.
